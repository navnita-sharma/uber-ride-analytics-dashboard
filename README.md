The Uber Ride Analytics Dashboard is an interactive Power BI project designed to provide insights into ride booking performance, revenue generation, customer experience, and operational efficiency.

Key metrics include:
• 93K completed rides
• 57K lost bookings
• 52M total revenue
• 2.51M total distance covered
• 24.64 average trip distance

The dashboard enables analysis of monthly booking trends, vehicle-wise revenue contribution, quarterly booking values, top pickup and drop locations, customer ratings, and driver ratings. Built using Power BI, Power Query, and DAX, the project demonstrates data cleaning, KPI tracking, business intelligence reporting, and interactive data visualization skills.
